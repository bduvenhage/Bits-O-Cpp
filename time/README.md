To compile, check out the repo and do:

```console
cd Bits-O-Cpp/time
mkdir build
cd build
cmake -D CMAKE_BUILD_TYPE=Release ..
make
```

Then run:

```console
./main
```

To run the C++11 std::chrono::high_resolution_clock example:
```console
./main_alt
```
